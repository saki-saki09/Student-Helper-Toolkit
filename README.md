# 🎓 Student Helper Toolkit

A simple but powerful Python CLI toolkit designed for students to automate daily tasks.

## 🚀 Features

- 🧮 Calculator (Basic + Expression Mode)
- 📁 File Renamer (Bulk rename files)
- 📂 File Organizer (Auto-sort files by type)
- 📊 Graph Generator (Quick plotting tool)

## 🎨 Highlights

- Clean CLI UI with colors
- ASCII logo interface
- Demo mode for quick testing
- Modular project structure

## 💡 Demo Mode

Just press Enter when asked for folder path to use built-in demo files.

## 📁 Project Structure

• modules/ → core features

• utils/ → helper functions

• assets/ → logo and demo files
